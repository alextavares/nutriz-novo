# _AI_TEAM (gestão como código) — Nutriz

Esta pasta é o “quartel-general” do app Nutriz (Flutter).
Aqui ficam:
- backlog único (`KANBAN.md`)
- decisões registradas (`DECISIONS.md`)
- eventos/métricas (`EVENTS.md`)
- processo (como trabalhar) (`PROCESS.md`)
- briefs e prompts por “expert” (`personas/*.md`)

Regra de ouro:
- Uma conversa -> um **brief** (`personas/00_BRIEF_TEMPLATE.md`) -> vira **tarefas** no `KANBAN.md` -> e, se for decisão relevante, entra em `DECISIONS.md`.
